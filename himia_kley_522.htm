    Клей 522

    Клей для линолеума и плитки ПВХ Forbo - универсальный дисперсионный клей высокого качества, который применяют для приклеивания различных покрытий ПВХ на предварительно подготовленные основания. Наносится при помощи зубчатых шпателей А1, А2 или В1.

    Eurosafe Star Tack

    Forbo 522 Eurosafe Star Tack
    универсальный клей для коммерческого линолеума

    Клей Форбо 522 - самый качественный, потому, самый популярный среди профессионалов клей для линолеума и плитки ПВХ. Клей Forbo (Форбо) 522 предназначен для напольных покрытий коммерческого назначения.

    Это высококачественный клей для напольных покрытий для приклеивания их на подготовленные абсорбирующие и неабсорбирующие основания.

    расход:

    Для шпателя A1/A2: 220-270 г/м2
    Для шпателя B1: 430-450 г/м2

    упаковка:

    Ведро пластиковое 13/20 кг, на поддоне — 44/32 шт.

    способ нанесения:

    Зубчатый шпатель А1/А2/B1 (в зависимости от типа покрытия)

    время подсушки:

    Для впитывающий оснований: 10-20 минут
    Для невпитывающих оснований: 30-90 минут

    время укладки:

    10-90 минут (в зависимости от типа основания)
