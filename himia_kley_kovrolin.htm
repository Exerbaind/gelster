    Клей для ковролина

    Коммерческий ковролин требует полной фиксации к основанию. Для приклеивания ковровых покрытий, заводом Форбо были разработаны специальные клеевые составы для ковролина.

    425
    Euroflex Standard Polaris

    Forbo 425 Euroflex Standard Polaris
    морозоустойчивый влажный клей

    Клей для ковролина Forbo (Форбо) 425 – это морозоустойчивый готовый клей для абсорбирующих оснований с высокой начальной и конечной клеящей способностью.

    Предназначен для укладки различных ПВХ и виниловых покрытий в рулонах, текстильных и ковровых покрытий (ковролина) на различных подложках, а также для иглопробивных покрытий.

    После размораживания клей восстанавливает свои клеевые характеристики. Содержит низкий процент растворителей <5%.
    Время для укладки 20 минут.

    расход:

    Для шпателя A2: 320 г/м2
    Для шпателя B1: 420-480 г/м2

    упаковка:

    Ведро — 13/20 кг, на поддоне — 44/32 шт.

    способ нанесения:

    Зубчатый шпатель А2/В1 (в зависимости от типа покрытия)

    время подсушки:

    10-20 минут

    время для укладки:

    ~ 20 минут

    скачать сертификат
    525
    Eurosafe Basic

    Forbo 525 Eurosafe Basic
    клей для ковролина

    Клей фиксатор Форбо 545 Полярис прекрасно подходит для фиксации ковровой плитки, текстильных покрытий, вдобавок, данный фиксатор станет хорошим решением для укладки плитки ПВХ, а также виниловых покрытий.
